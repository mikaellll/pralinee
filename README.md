# 🥐 Pralinée - L'Art de la Pâtisserie Française

Bienvenue sur **Pralinée**, une plateforme dédiée à l'excellence de la boulangerie et de la pâtisserie française. Ce projet combine passion culinaire et technologies web modernes pour offrir une expérience de partage de recettes professionnelle, pédagogique et bilingue.

---

## ✨ Caractéristiques

- 🌍 **Bilingue (FR/EN)** : Navigation et contenus intégralement traduits pour une portée internationale.
- 🥖 **Recettes Pédagogiques** : Chaque article inclut des fiches techniques, des méthodes pas à pas détaillées et des conseils de chef.
- 🚀 **Performance Maximale** : Construit avec **Astro 5.0**, garantissant des temps de chargement ultra-rapides et un score Lighthouse optimal.
- 🎨 **Design Raffiné** : Une interface moderne et épurée utilisant **Tailwind CSS**, optimisée pour le confort de lecture.
- 🔍 **SEO de Pointe** : Structure sémantique, métadonnées complètes et sitemap automatique pour une visibilité optimale sur les moteurs de recherche.

---

## 🛠️ Stack Technique

- **Framework** : [Astro 5.0](https://astro.build/)
- **Styling** : [Tailwind CSS](https://tailwindcss.com/)
- **Contenu** : Markdown & MDX (Content Collections)
- **Déploiement** : Cloudflare Pages / Vercel
- **Images** : Astro Assets pour une optimisation automatique

---

## 🚀 Démarrage Rapide

### Installation

```shell
npm install
```

### Développement

Lancer le serveur local sur `localhost:4321` :

```shell
npm run dev
```

### Build Production

```shell
npm run build
```

## 📂 Structure du Projet

```text
/
├── src/
│   ├── assets/       # Images et styles
│   ├── components/   # Composants UI et Widgets
│   ├── content/      # Configuration des collections
│   ├── data/         # Articles MD par langue (post/ et post/en/)
│   ├── layouts/      # Modèles de pages
│   └── pages/        # Routes et pages bilingues
├── public/           # Assets statiques
└── astro.config.ts   # Configuration principale
```
